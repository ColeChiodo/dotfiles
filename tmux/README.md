# tmux

![TMUX Example](../.resources/tmux.png)

| Setting | Value |
|---------|-------|
| Prefix | `Ctrl+Space` |
| Mouse | On |
| Passthrough | On |
| Prev/next window | `j` / `k` |
| Status bar | Top, tabs only |
| Base index | 1 |
| Window numbers | 1-based |
| Colorscheme | Gruvbox-dark inspired |
